<h1>FeatureHub: Datasets</h1>

## What Are FeatureHub Datasets?

FeatureHub datasets are curated, production-ready datasets designed to plug directly into modern quantitative research workflows.

## UI Access

FeatureHub Datasets are accessible through the ForecastOS UI, where users can browse available datasets and associated metadata and analytics.

![FeatureHub Datasets navigation](/screenshot_dataset.png)

## API Access

FeatureHub Datasets are also available programmatically via the ForecastOS API.

Clients with API access can retrieve datasets directly for integration into internal research pipelines, backtesting systems, and production workflows.

Detailed endpoint documentation and examples are available to clients with active API access. If you’re evaluating FeatureHub and would like more information, the ForecastOS team is happy to help.

## Open-Source Access

For teams working in Python, FeatureHub Datasets can be accessed through ForecastOS’s open-source tooling, designed to integrate cleanly with pandas-based research workflows.

Full dataset access methods and advanced usage details are available to clients with FeatureHub access.
